## Project Description
- Office Hours Scanner serves as a tool for university/college students to get an organized summary of the office hours of their courses within seconds
- OpenAI Assistant is integrated to accurately identify and summarize the office hours
- Office Hours Scanner automatically removes user's uploaded syllabus files after the summary is presented
- Proudly served 50+ users since launch in September 2024

## Technology Stack
- NextJS
- OpenAI Assistant
- REST API
- Vercel

## Web App Usage Instructions
- Upload syllabus files from device
- Users can select single or multiple files simultaneously
- Users can remove mistakenly uploaded files
- Scan Office Hours and wait for result

## Repo Clone Usage Note
- Users must create their own OpenAI apiKey to have their own version of the web app
